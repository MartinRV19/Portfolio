Weather Data Analysis

This notebook analyzes weather data from our weather dashboard application.

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

# Set the style for our plots
sns.set(style="whitegrid")

Loading the Dataset

First, we'll load our weather data.

# Sample data (in a real notebook, we would load from a file)
data = {
    'Date': ['2023-01-01', '2023-01-02', '2023-01-03'],
    'City': ['New York', 'Los Angeles', 'Chicago'],
    'Temperature': [32, 75, 28],
    'Humidity': [65, 45, 70],
    'Precipitation': [0.1, 0.0, 0.3]
}

df = pd.DataFrame(data)
df

Temperature Analysis

Let's analyze the temperature differences between cities.

# Group by city and calculate average temperature
city_temps = df.groupby('City')['Temperature'].mean()

Conclusion

Based on our analysis, Los Angeles has the highest temperature at 75°F, while Chicago has the lowest at 28°F. New York falls in between at 32°F. This data can help users plan their travel or activities based on weather conditions in different cities.
